# Baozi Store API

API desenvolvida com Spring Boot para gerenciamento de:

- Clientes
- Produtos
- Pedidos

## Tecnologias
- Java
- Spring Boot
- Spring Data JPA
- H2 Database

## Endpoints

### Cliente
- POST /clientes
- GET /clientes
- GET /clientes/{id}
- DELETE /clientes/{id}

### Produto
- POST /produtos
- GET /produtos
- GET /produtos/{id}
- DELETE /produtos/{id}

### Pedido
- POST /pedidos
- GET /pedidos
- GET /pedidos/{id}
- DELETE /pedidos/{id}
